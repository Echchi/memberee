# 💫 memberee
### 누구나 편하게 쓰기 쉬운 회원관리
## 핵심 라이브러리
- tailwindcss : css in js 구현
- axios : 리퀘스트
- eslint : 코드 품질 유지
- prettier : 코드 품질 유지
- redux : 글로벌 저장소

<hr/>

## 시작 방법

### `npm install`

필요한 라이브러리를 설치한다.

### `next dev`
개발 서버를 실행한다.
